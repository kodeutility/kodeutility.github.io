There are 3 types of numbers in python - integer, float, complex. They are defined as int, float, complex class in python respectively.

Integes are numbers which do not have decimal point.
Float are numbers which have decimal point.
Complex are numbers which have real part and imaginary part in the form of x+yj, where 'x' is real part and 'y' is imaginary part

We can find the type of a variable using the in-built function type().

int

Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.

test.py
x = 50
y = 0
z = -100

print(type(x))
print(type(y))
print(type(z))
Output
<class 'int'>
<class 'int'>
<class 'int'>

float

Float, or "floating point number" is a number, positive or negative, containing one or more decimals. Float can also be scientific numbers with an "e" to indicate the power of 10. While integers can be of any length, a floating point number is accurate only up to 15 decimal places (the 16th place is inaccurate).

test.py
x = 50.
y = 0.053
z = -100.123e3

print(type(x))
print(type(y))
print(type(z))
Output
<class 'float'>
<class 'float'>
<class 'float'>

complex

A complex number is a Python number type made of real and imaginary parts. It is represented as x+yj. To denote the irrational part we can’t use the letter ‘i’, like you would do on paper. Also, it is mandatory to provide a coefficient to the imaginary part.

test.py
x = 6+5j
y = 0j
z = -3j

print(type(x))
print(type(y))
print(type(z))
Output
<class 'complex'>
<class 'complex'>
<class 'complex'>

Expressing numbers in binary, octal, hexadecimal

Numbers we deal with everyday are decimal (base 10) number system. But computer programmers (generally embedded programmer) need to work with binary (base 2), hexadecimal (base 16) and octal (base 8) number systems.
In Python, we can represent these numbers by appropriately placing a prefix before that number.

Number System Prefix
Binary '0b' or '0B'
Octal '0o' or '0O'
Hexadeciaml '0x' or '0X'
test.py
# Output: 107
print(0b1101011)

# Output: 253 (251 + 2)
print(0xFB + 0b10)

# Output: 13
print(0o15)
Output
107
253
13
