Python is a high level programming language. It was created by Guido Van Rossum in 1991.

Python Uses:

  • Web development
  • Scientific computing
  • Desktop GUI applications
  • Software development
  • Business applications
  • Scripting
  • Game development
  • Machine Learning/ AI
  • Natural language processing (Chatbots)

What are Python features?

  • Easy to learn,read,maintain
  • Interpreted language:Execution of code is line by line at a time.
  • Cross-platform: available for windows,mac,linux
  • Open Source:Source code is freely available
  • Object oriented: Supports concepts such as Objects and classes
  • Extensive Library support. Hence it's refered as "Batteries included"
  • Can be integrated with other programming languages like C,C++,Java
  • It supports multiple programming paradigms - Functional, Object Oriented, Imperative, Procedural
  • Robust - exception handling is available, memory management techniques is in-built
  • Emphasis is provided on code readability and less lines of code

There are two versions of python available.

  • Python 2
  • Python 3

Use python 3 because python 2 support will be stopped by 2020.

Download & Install Python at www.python.org/downloads.
