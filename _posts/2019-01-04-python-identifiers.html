Identifier is the name given to entities like class, functions, variables etc. in a programming language. It helps differentiating one entity from another.

Rules for writing identifiers:

  • Identifiers can be a combination of letters in lowercase (a to z) or uppercase (A to Z) or digits (0 to 9) or an underscore ' _ '
  • An identifier cannot start with a digit
  • Keywords cannot be used as identifiers
  • No special characters ( !, @, #, $, %, ^, &, *, ~ ) are allowed except underscore ' _ ' for naming an identifier
  • Identifier can be of any length. No limit
  • Python language is case sensitive i.e python , Python, PYTHON are considered as different identifiers
  • Use capital letters where possible to declare a constant.
  • Constants are put into Python modules and meant not be changed.

TIP

  • Name your variable that make sense to the task it does.
  • If there are multiple words use underscore to separate each word
