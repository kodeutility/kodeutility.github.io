Operators are special symbols in Python that carry out arithmetic or logical computation. The value that the operator operates on is called the operand.
E.g. 2+3 , Here 2 and 3 are operands and + is the operator.
There are 7 types of operators.

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operators
  • Assignment Operators
  • Identity Operators
  • Membership Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication etc.

Operator Meaning Example
+ Add two operands or unary plus 3+2
- Subtract right operand from the left or unary minus 3-2
* Multiply two operands 3*4
/ Divide left operand by the right one (always results into float value) 3/4
% Modulus - remainder of the division of left operand by the right 4%3 (raminder of 4/3)
// Floor division - division that results into whole number adjusted to the left in the number line 4//3
** Exponent - left operand raised to the power of right 2**4 (2 to the power 4)
test.py
num_1 = 4
num_2 = 3

# Output: num_1 + num_2 = 7
print('num_1 + num_2 =',num_1+num_2)

# Output: num_1 - num_2 = 1
print('num_1 - num_2 =',num_1-num_2)

# Output: num_1 * num_2 = 12
print('num_1 * num_2 =',num_1*num_2)

# Output: num_1 / num_2 = 1.333333
print('num_1 / num_2 =',num_1/num_2)

# Output: num_1 // num_2 = 1
print('num_1 // num_2 =',num_1//num_2)

# Output: num_1 ** num_2 = 64
print('num_1 ** num_2 =',num_1**num_2)
Output
num_1 + num_2 = 7
num_1 - num_2 = 1
num_1 * num_2 = 12
num_1 / num_2 = 1.3333333333333333
num_1 // num_2 = 1
num_1 ** num_2 = 64

Relational/Comparison Operators

Comparison operators are used to compare values. It either returns True or False according to the condition.

Operator Meaning Example
> True if left operand is greater than the right 3>4
< True if left operand is less than the right 3<4
== True if both operands are equal 3==4
!= True if operands are not equal 3!=4
>= True if left operand is greater than or equal to the right 3>=4
<= True if left operand is less than or equal to the right 3<=4
test.py
num_1 = 10
num_2 = 12

# Output: num_1 > num_2 is False
print('num_1 > num_2  is',num_1>num_2)

# Output: num_1 < num_2 is True
print('num_1 < num_2  is',num_1= num_2 is False
print('num_1 >= num_2 is',num_1>=num_2)

# Output: num_1 <= num_2 is True
print('num_1 <= num_2 is',num_1<=num_2)
Output
num_1 > num_2  is False
num_1 < num_2  is True
num_1 == num_2 is False
num_1 != num_2 is True
num_1 >= num_2 is False
num_1 <= num_2 is True

Logical Operators

Logical operators are the and, or, not operators.

Operator Meaning Example
and True if both the operands are true x and y
or True if either of the operands is true x or y
not True if operand is false (complements the operand) not x
test.py
var_1 = True
var_2 = False

# Output: var_1 and var_2 is False
print('var_1 and var_2 is',var_1 and var_2)

# Output: var_1 or var_2 is True
print('var_1 or var_2 is',var_1 or var_2)

# Output: not var_1 is False
print('not var_1 is',not var_1)
Output
var_1 and var_2 is False
var_1 or var_2 is True
not var_1 is False

Bitwise Operators

Bitwise operators act on operands as if they were string of binary digits. It operates bit by bit, hence the name.
E.g. 2 is 10 in binary and 7 is 111.

Let x = 10 (0000 1010 in binary) and y = 4 (0000 0100 in binary)

Operator Meaning Example
& Bitwise AND x& y = 0 (0000 0000)
| Bitwise OR x | y = 14 (0000 1110)
~ Bitwise NOT ~x = -11 (1111 0101)
^ Bitwise XOR x ^ y = 14 (0000 1110)
>> Bitwise right shift x>> 2 = 2 (0000 0010)
<< Bitwise left shift x<< 2 = 40 (0010 1000)

Assignment Operators

Assignment operators are used in Python to assign values to variables.
num_1 = 5 is a simple assignment operator that assigns the value 5 on the right to the variable num_1 on the left.
There are various compound operators in Python like num_1 += 5 that adds to the variable and later assigns the same. It is equivalent to num_1 = num_1 + 5.

Operator Meaning Example
= x = 5 x = 5
+= x += 5 x = x + 5
-= x -= 5 x = x - 5
*= x *= 5 x = x * 5
/= x /= 5 x = x / 5
%= x %= 5 x = x % 5
//= x //= 5 x = x // 5
**= x **= 5 x = x ** 5
&= x &= 5 x = x & 5
|= x |= 5 x = x | 5
^= x ^= 5 x = x ^ 5
>>= x >>= 5 x = x >> 5
<<= x <<= 5 x = x << 5

Special Operators

is and is not are the identity operators in Python. They are used to check if two values (or variables) are located on the same part of the memory. Two variables that are equal does not imply that they are identical.

Operator Meaning Example
is True if the operands are identical (refer to the same object) x is True
is not True if the operands are not identical (do not refer to the same object) x is not True
test.py
x_1 = 5
y_1 = 5
x_2 = 'Hello'
y_2 = 'Hello'
x_3 = [1,2,3]
y_3 = [1,2,3]

# Output: False
print(x_1 is not y_1)

# Output: True
print(x_2 is y_2)

# Output: False
print(x_3 is y_3)
Output
False
True
False

x_1 and y_1 are integers of same values, so they are equal as well as identical. Same is the case with x_2 and y_2 (strings).
But x_3 and y_3 are list. They are equal but not identical. It is because interpreter locates them separately in memory although they are equal.

Membership Operators

in and not in are the membership operators in Python. They are used to test whether a value or variable is found in a sequence (string, list, tuple, set and dictionary).

Operator Meaning Example
in True if value/variable is found in the sequence 5 in x
not in True if value/variable is not found in the sequence 5 not in x
test.py
x = 'Hello world'
y = {1:'a',2:'b'}

# Output: True
print('H' in x)

# Output: True
print('hello' not in x)

# Output: True
print(1 in y)

# Output: False
print('a' in y)
Output
True
True
True
False
